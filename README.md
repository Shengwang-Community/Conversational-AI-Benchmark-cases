# 声网AI模型评测平台（对话式）

🌐 [Website](https://www.shengwang.cn/duihua/benchmark)

## 简介

此仓库包含声网AI模型评测平台（对话式）质量指标的模型测试集。

### TTS

| 文件夹  | 简介                                   |
| ------- | -------------------------------------- |
| TTS-AP  | 多种语言维度在特定复杂场景的测试文本。 |
| TTS-WER | 多种语言维度的基础测试文本             |
